# Internal Notes

This directory contains planning notes, milestone handoffs, validation fixtures, and QA checklists that are useful for development but are not the public front door of the repository.

Notable archived material here includes:

- `Roadmap-internal.md`: the longer milestone-oriented working roadmap that previously lived in the repo root
- dated milestone handoffs (`omd-milestone*.md`)
- validation and parity checklists used during UI and renderer work

The root of the repo is now reserved for user-facing docs such as:

- [README.md](../../README.md)
- [Roadmap.md](../../Roadmap.md)
- [OpenIssues.md](../../OpenIssues.md)
- [ClosedIssues.md](../../ClosedIssues.md)
- [WINDOWS_BUILD.md](../../WINDOWS_BUILD.md)

If a document here later becomes useful to users or contributors, it can be promoted back into a public-facing doc with a cleaner rewrite.
